深入理解javascript之typeof和instanceof

typeof

typeof方法返回一个字符串，来表示数据的类型。

各个数据类型对应typeof的值：

数据类型 Type
Undefined “undefined”
Null “object”
Boolean “boolean”
Number “number”
String “string”
Symbol “symbol”
宿主对象(JS环境提供的，比如浏览器) Implementation-dependent
函数对象Function “function”
任何其他对象Object “object”
